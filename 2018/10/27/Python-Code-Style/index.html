Python Code Style

风格规范

缩进

用4个空格来缩进代码 ### 空行 顶级定义之间空两行, 比如函数或者类定义.
方法定义, 类定义与第一个方法之间, 都应该空一行. ### 空格 括号内不要有空格
逗号, 分号, 冒号后面加空格（行尾除外），前面不加空格
索引或切片的左括号前不应加空格
在二元操作符两边都加上一个空格, 比如赋值(=), 比较(==, <, >, !=, <>, <=, >=, in, not in, is, is not), 布尔(and, or, not)
算术操作符两边是否有空格应保持一致
参数的=不加空格
